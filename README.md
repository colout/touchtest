# TouchTest - MacOS AirHorn
Based off of: https://github.com/mrmekon/touchtest

## Usage

### Install as Persistent Daemon
```
$ make install
```

### Uninstall Daemon
```
$ make uninstall
```

### Build
```
$ make
```

### Build and launch
```
$ make run
```

### Kill running daemon
```
$ make kill
```

### Cleanup
```
$ make clean
```
